## Overview

CodeRev is a modern, AI-enhanced code editor that combines real-time collaboration with intelligent coding assistance. Built with Next.js and powered by Google's Gemini AI, it offers a seamless coding experience with features like smart code completion, documentation generation, and multi-language support including C/C++.

## üì¶ Installation

1. **Clone the repository:**

```bash
git clone https://github.com/aniket-patill/CodeRev---AI-Powered-Online-Code-Editor.git
```

2. **Install dependencies:**

```bash
cd CodeRev---AI-Powered-Online-Code-Editor
npm install
```

3. **Set up environment variables:**
   Create a `.env.local` file with:

```env
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key
```

4. **Run the development server:**

```bash
npm run dev
```
